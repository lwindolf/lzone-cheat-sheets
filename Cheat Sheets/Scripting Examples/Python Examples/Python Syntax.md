### Reading from a Pipe in Python

Line by line

    proc = subprocess.Popen(['/bin/ls', 'something'],stdout=subprocess.PIPE)
    while True:
      line = proc.stdout.readline()
      if line != '':
        do_something
      else:
        break

Entire output at once

    echo subprocess.check_output(['/bin/ls', 'something])
