## CLI

    gpupdate /force                # Force evaluation now
    
    gpresult /r                    # Report all configured GPOs 
    gpresult /h C:\gpo.html        # Export all configured GPOs in HTML file
    
    gpresult /user <username> /v   # Check which GPOs apply to user

Also check the [Powershell GPO cmdlets](https://learn.microsoft.com/en-us/powershell/module/grouppolicy/get-gpo?view=windowsserver2025-ps)
